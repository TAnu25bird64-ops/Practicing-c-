# Bomb simulation

"."   : Empty space (空きマス)
"#"  : Wall (壁)
"B"  : Bomb (爆弾)

爆弾は，上下左右に爆風を飛ばす
条件としては以下を満たす:

・#にあたると，その方向の爆風は止まる
・爆風の届いたマスを*にする
・爆弾自身のマスも*にする
・爆弾の長さは"K"

入力形式は，以下のようになる
H W K  
(マップの形式)

* Hは行，Wは列，そしてKはマップの長さを表す

C++で作成した簡素な爆風シミュレーションです

##使用技術
-プログラミング言語: C++
-二次元グリッド
-シミュレーション
-vector<string>
-dx dy 移動配列

##仕様
-B : 爆弾
-# : 壁
-* : 爆風範囲
